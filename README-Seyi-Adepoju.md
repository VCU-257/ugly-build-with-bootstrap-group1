## 1. My Assigned Work

Built the **Confirmation Page** for the Budgetly application. This page gives users feedback after completing an action and confirms that their information or submission was successful.

## 2. DaisyUI Implementation

- **Components Used:**  
  Used DaisyUI components such as **cards**, **buttons**, and alert/success styling to create a clean and user-friendly confirmation page. I also used spacing and layout utilities to keep the page responsive and visually organized.

One challenge was making the confirmation page feel clear and polished while still matching the overall design of the application. I focused on creating a simple layout, readable text hierarchy, and clear navigation so users would immediately understand that their action was completed successfully.
